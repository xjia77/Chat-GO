# Chat-GO Project
```
Use Go's WebSocket-based communication chat software.
```
# Function
```
Sign In Register
Modify Avatar
Group Chat
Group Friends List
Single Chat
Add Friends
Add Group
Text Message
Clipboard Images
Picture Messaging
File Sending
Voice Messages
Video messaging
Screen sharing (picture-based)
Video calling (p2p video calling based on WebRTC)
Distributed deployment (unified messaging via kafka global message queue, which allows horizontal scaling of the system)
```
# Front-end technologies and frameworks
```
React
Redux state management
AntDesign
Use of proto buffer
WebSocket
Clipboard file reading and manipulation
Chat box to send text to the bottom of the display
FileReader for file manipulation
Conversion between ArrayBuffer, Blob, Uint8Array
Get camera video (mediaDevices)
Get microphone audio (Recorder)
Get screen share (mediaDevices)
WebRTC p2p video call
```
